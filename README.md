# Проект CityGuru
CityFinder — это мобильное приложение для быстрого и удобного поиска информации о городах. Оно помогает пользователям мгновенно находить нужный город из обширного списка, что удобно при планировании путешествий, проверке географии или просто для расширения кругозора.

Многие стандартные списки городов в приложениях неудобны, особенно когда список большой. Это приложение решает проблему с помощью быстрого, отзывчивого поиска с мгновенными результатами.

Главной целью была практика и закрепление современных подходов к разработке под Android. В частности, я упражнялась с библиотекой Compose для построения UI и стремилась создать чистейшую архитектуру MVVM.

Язык: Kotlin
UI: Jetpack Compose
Архитектура: MVVM (Model-View-ViewModel)
Асинхронность: Kotlin Coroutines + Flow
DI: Koin
